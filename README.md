# 小红书选题器

一个本地保存的 Chrome/Edge 浏览器插件原型，用来采集小红书笔记的封面、标题、关键词和选题内容，并按栏目整理成表格。

## 功能

- 在小红书笔记页点击插件，自动提取标题、封面、链接和正文摘要。
- 根据标题、正文和话题标签生成一组本地关键词、封面共性、内容共性和选题方向。
- 可填写 DeepSeek API Key，用 AI 分析标题、正文和封面链接，生成更像人工拆解的结果。
- 采集结果保存到本地浏览器存储；只有点击“AI 分析”时才会把当前笔记内容发送给 DeepSeek。
- 弹窗顶部提供常用操作：AI 分析、保存、重抓，不用拖到最下面。
- 弹窗关闭后会保留当前页面的分析草稿，直到刷新或关闭该页面。
- 支持在小红书页面右键，一键“AI 分析并保存到选题器”。
- 支持自定义 AI 分析重点、输出长度和额外 Prompt。
- 支持按设置的间隔检查 GitHub Release 是否有新版。
- 检查到新版时会弹浏览器通知，确认后打开下载页和扩展管理页。
- 按“绘画教程”“稿件展示”等栏目归档。
- 在表格库里搜索、筛选、删除、导出 CSV。
- 支持新增自定义栏目。

## 安装

1. 打开 Chrome 或 Edge 的扩展管理页。
2. 开启“开发者模式”。
3. 选择“加载已解压的扩展程序”。
4. 选择本项目文件夹：`D:\Onedrive\文档\选题器`。

## 使用

1. 打开一篇小红书笔记页面。
2. 点击浏览器工具栏里的“小红书选题器”。
3. 选择栏目，比如“绘画教程”或“稿件展示”。
4. 检查标题、关键词、选题内容和备注。
5. 如果已经在表格库里设置 API Key，可以点击“AI 分析”生成更完整的拆解。
6. 点击“保存到本地”。
7. 点击弹窗右上角“表”进入表格库。

## 右键一键保存

1. 打开一篇小红书笔记页面。
2. 在页面空白处或内容上右键。
3. 选择“AI 分析并保存到选题器”。
4. 插件会自动分析并保存到本地表格库。

右键保存成功时，扩展图标会短暂显示 `OK`；失败会显示 `ERR`。失败时通常是没有保存 DeepSeek API Key、模型名不可用、网络请求失败，或当前页面不是小红书笔记页。

## AI 设置

1. 点击插件弹窗右上角“表”，进入表格库。
2. 在“AI 设置”里填写 DeepSeek API Key。
3. 模型默认是 `deepseek-v4-flash`，也可以手动改成 `deepseek-v4-pro`。
4. 可以调整输出长度、分析重点和自定义 Prompt；默认 Prompt 会直接显示在设置页里，可以编辑，也可以点“恢复默认 Prompt”复位。
5. 回到小红书笔记页，点击插件里的“AI 分析”。

API Key 保存在浏览器扩展的本地存储里。这个版本适合私下自用，不建议直接发布到公开扩展商店。点击“AI 分析”时，插件会把当前笔记的标题、正文摘要、链接、标签和封面链接发送给 DeepSeek。

## 更新检查

插件可以定期检查 GitHub Release：

1. 把插件代码放到 GitHub 仓库。
2. 每次更新时发布一个新的 Release，tag 使用版本号，比如 `v0.2.0`、`v0.3.0`。
3. 在表格库的“更新检查”里填写 GitHub 用户名、仓库名和检查间隔。
4. 有新版时，扩展图标会显示 `NEW`，设置页会显示 Release 和 zip 下载链接。
5. 插件会弹出通知，点击“下载新版”会打开 zip/Release 链接，并打开扩展管理页。

浏览器不允许“加载已解压扩展程序”静默覆盖自身文件，所以这个版本不能真正自动安装更新。你们仍然需要下载 zip、解压覆盖插件文件夹，然后在扩展管理页点“重新加载”。如果以后想完全自动更新，需要改成 Chrome Web Store 发布，或制作带 `update_url` 的自托管 `.crx` 更新通道。

## Windows 一键更新

从 `v0.2.2` 开始，插件文件夹里包含：

- `update.bat`
- `update.ps1`

以后如果检查到新版，可以直接在插件文件夹里双击 `update.bat`。它会自动：

1. 检查 `AstroloNeko/xhs-topic-picker` 的最新 GitHub Release。
2. 下载 Release 里的 zip 附件。
3. 解压并覆盖当前插件文件夹。
4. 打开扩展管理页，提示你点击“重新加载”。

浏览器仍然不会允许插件自己运行本地程序，所以这一步需要你手动双击 `update.bat`。如果 Windows 弹出安全提示，选择允许运行即可。

## 本地存储位置

表格数据保存在浏览器扩展的本地存储里。普通浏览器插件不能静默选择并写入任意本地文件夹，所以暂时不能设置一个固定电脑目录来实时保存表格文件。需要文件时可以在表格库点击“导出 CSV”。

## 后续可升级方向

- 接入 AI：自动总结爆款共性、封面结构、标题公式。
- 批量采集：一次性分析多个链接，提取共性词和高频选题。
- 图片本地缓存：把封面图片保存为离线资源。
- 数据分组：给不同账号、赛道、内容类型单独建库。
